## `mv`

One way to use `mv` command is to change the file name.
```sh
$ mv foo.py foo.rb
```
This will change `foo.py` to `foo.rb`.

Here's a little shortcut to it.
```sh
$ mv foo.{py, rb}
```
This produces the same result.
