## [Lv.1 자릿수 더하기](https://programmers.co.kr/learn/courses/30/lessons/12931)

시간 복잡도: **O(N)**

<br> 

```cpp
#include <iostream>
using namespace std;

int solution(int n)
{
  int ret=0;
  while(n) {
    ret += n%10;
    n/=10;
  }

  return ret;
}
```

<br>

`N`이 0이 될때까지 마지막 숫자를 잘라서 더하는 방식.
