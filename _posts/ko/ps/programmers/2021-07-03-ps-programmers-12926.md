## [Lv.1 시저 암호](https://programmers.co.kr/learn/courses/30/lessons/12926)

시간 복잡도: **O(n)**

```cpp
#include <string>
using namespace std;

string solution(string s, int n) {
  const int SIZE = s.size();

  for(int i=0; i<SIZE; ++i) {
    if(s[i]==' ') continue;
    if(islower(s[i])) {
      s[i] = 97 + ((s[i]-97+n)%26);
    } else if(isupper(s[i])) {
      s[i] = 65 + ((s[i]-65+n)%26);
    }
  }

  return s;
}
```

<br>

소문자와 대문자는 각각 `islower()` 함수와 `isupper()`함수를 사용해서 구했다.
