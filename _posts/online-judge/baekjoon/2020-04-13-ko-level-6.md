### [2577. 숫자의 개수](https://www.acmicpc.net/problem/2577) <a id="2577"></a>
```rb
a = gets.to_i
b = gets.to_i
c = gets.to_i
t = [0]*10
a = a*b*c
while a > 0
    t[a%10]+=1
    a/=10
end

puts t
```

### [1546. 평균](https://www.acmicpc.net/problem/1546) <a id="1546"></a>
```rb
n = gets.to_i

max = g.max
*g = g.map{|i| (i*100)/max}
puts g.sum / g.size
```

### [8958. Score](https://www.acmicpc.net/problem/8958) <a id="8958"></a>
```rb
t = gets.to_i
while t>0
    score = 0
    pt = 1
    str = gets.chomp
    for i in 0...str.size
        if str[i] == 'O'
            score += pt
            pt += 1  
        else
            pt = 1
        end
    end
    puts score
    t -= 1
end
```

### [4344. Above Average](https://www.acmicpc.net/problem/4344) <a id="4344"></a>
```rb
t = gets.to_i
while t > 0
    c, *g = gets.split.map(&:to_i)
    avg = g.sum / c
    printf('%.3f', ((g.select{|x| x>avg}).size)*100.0/c)
    puts "%"
    t -= 1
end
```
